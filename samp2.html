Title : Obstacle Detection Using IR Sensor

Synopsis:In this IR proximity sensor system, the distance from the sensor to the object is indicated by 4 LEDs. It is calculated in terms of the voltage obtained by the IR sensor at different object distances and compare these values with 4 reference values. That is 4 distance point represents 4 voltage values. The 4 LEDs will ON one by one, as the object cross each distance points.
